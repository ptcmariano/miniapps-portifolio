Privacidade

Última atualização: Fevereiro de 2026

O que coletamos?

Absolutamente nada de pessoal. Não solicitamos e-mail, nome, telefone ou qualquer informação que possa identificar você. Nossos mini-aplicativos funcionam de forma totalmente anônima.

Onde estão meus dados?

Seus dados (como treinos, horários favoritados ou históricos de partidas) são salvos exclusivamente no seu dispositivo, usando uma tecnologia chamada localStorage. Nós não temos acesso a esses dados e eles nunca saem do seu celular ou computador. Se você limpar os dados do seu navegador, esses dados serão excluídos.

Analytics

Utilizamos ferramentas de análise (como Google Analytics ou similar) para entender como o site é usado (quais apps são mais clicados, tempo de permanência, etc.). Esses dados são agregados e anônimos, servindo apenas para melhorar a ferramenta.

Links Externos

Alguns de nossos aplicativos podem conter links para o GitHub ou WhatsApp. Ao clicar neles, você estará sujeito às políticas de privacidade dessas plataformas.

Dúvidas?

Se você tiver qualquer dúvida sobre como tratamos a privacidade, pode entrar em contato via WhatsApp.
